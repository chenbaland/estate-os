**Format:** `As a [persona], I want [goal], so that [benefit].`

## Epic 1: Authentication & Authorization

### US-AUTH-001
**As a** resident, **I want to** log in with my email and password, **so that** I can access my estate dashboard securely.

**Story Points:** 3 | **Priority:** P0

### US-AUTH-002
**As a** resident, **I want to** log in with my phone number via OTP, **so that** I can access the app without remembering a password.

**Story Points:** 5 | **Priority:** P0

### US-AUTH-003
**As a** resident, **I want to** sign in with Google or Apple, **so that** I can register quickly with my existing account.

**Story Points:** 5 | **Priority:** P0

### US-AUTH-004
**As an** estate admin, **I want to** enable MFA on my account, **so that** my administrative access is protected from credential theft.

**Story Points:** 5 | **Priority:** P0

### US-AUTH-005
**As a** user, **I want to** view and revoke active sessions on other devices, **so that** I can secure my account if a device is lost.

**Story Points:** 3 | **Priority:** P0

### US-AUTH-006
**As a** super admin, **I want to** assign RBAC roles to users within an estate, **so that** each person has appropriate permissions.

**Story Points:** 8 | **Priority:** P0
